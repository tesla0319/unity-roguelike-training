## ダウンロードして遊ぶ (Windows)

> Unity のインストール不要です。ダウンロードしてすぐ遊べます。

### 手順

1. **[Releases](../../releases/latest)** を開く
2. `My.project_Windows.zip`（またはそれに相当するファイル）をダウンロード
3. zip を任意のフォルダに解凍する
4. 解凍したフォルダ内の **`My project.exe`** をダブルクリックして起動

```
📁 解凍後のフォルダ構成
├── My project.exe        ← これを起動
├── My project_Data/
├── MonoBleedingEdge/
└── UnityCrashHandler64.exe
```

> **注意**：`My project.exe` は必ず上記フォルダごと保持してください。  
> exe 単体をほかの場所に移動しても起動しません。

## 遊び方

### 操作

| キー | 操作 |
|---|---|
| `W` / `↑` | 上に移動 |
| `S` / `↓` | 下に移動 |
| `A` / `←` | 左に移動 |
| `D` / `→` | 右に移動 |
| `H` | 回復薬を使用 |
| `R` | リスタート（いつでも可） |

### ゲームルール

- **ターン制**：キー入力 1 回 = 1 ターン経過
- **戦闘**：敵がいるマスへ移動すると自動攻撃
- **目標**：`S`（階段）を探して 3F まで到達するとクリア
- **敗北**：HP が 0 になるとゲームオーバー

---

## マス記号の見方

| 記号 | 色 | 意味 |
|---|---|---|
| `P` | 青 | プレイヤー |
| `N` | 赤 | Normal 敵（標準） |
| `F` | 紫 | Fast 敵（素早い） |
| `T` | 濃赤 | Tank 敵（高耐久） |
| `H` | 緑 | 回復薬（乗ると自動取得） |
| `D` | オレンジ | ダメージ床（乗ると 5 ダメージ） |
| `S` | 黄 | 階段（乗ると次フロアへ） |

---

## 敵の特徴

| 種類 | HP | ATK | 特徴 |
|---|---|---|---|
| Normal | 30 | 10 | 標準的な追尾と攻撃 |
| Fast | 15 | 8 | 1 ターンに最大 2 マス追尾 |
| Tank | 60 | 15 | 高耐久・高火力、移動は標準 |

## 技術構成

| 項目 | 内容 |
|---|---|
| エンジン | Unity 6 |
| 言語 | C# |
| 座標管理 | `Vector2Int`（論理座標） + `Transform`（描画座標）分離 |
| 演出 | Coroutine ベース（スライド・点滅・撃破） |
| UI | TextMeshPro（ワールドスペース） + UGUI |
| マップ生成 | ランダム配置 + BFS 連結チェック + リトライ |

### 主要スクリプト

| スクリプト | 役割 |
|---|---|
| `GameManager` | 状態・ターン・スコア管理（Singleton） |
| `GridManager` | マップ生成・タイル管理・座標変換 |
| `FloorManager` | フロア遷移・敵・アイテム配置 |
| `PlayerController` | 入力受付・移動・戦闘・演出 |
| `EnemyController` | 敵 AI・移動・戦闘・演出 |
| `UIManager` | HUD・GameOver / Clear パネル更新 |
| `GameConfig` | 全パラメータ集約（マジックナンバー排除） |

## セットアップ（開発者向け）

ソースコードから動かす場合の手順です。遊ぶだけなら[こちら](#ダウンロードして遊ぶ-windows)。

1. このリポジトリをクローン
2. Unity 6 でプロジェクトを開く
3. `MainScene` を開いて Play

> TextMeshPro Essential Resources が未インポートの場合、Unity が自動でダイアログを表示します。`Import` を選択してください。
